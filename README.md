# Nobles Portfolio

Personal portfolio website of Jade Anthony T. Nobles. The site presents profile information, resume details, projects, practicum experience, and contact information.

## Pages

- Home: portfolio overview and skills
- Resume: education, skills, certificates, and experience
- Projects: featured academic and technical projects
- Practicum: company overview, nature of work, rendered hours, and conclusion
- Contact: email, phone, LinkedIn, and GitHub links

## Practicum

The practicum page documents the engagement at the City Government of Biñan. The practicum started on April 20, 2026. As of July 10, 2026, 400 of the required 486 hours are complete, with 86 hours remaining.

## Technologies

- HTML
- CSS
- Bootstrap
- JavaScript

## Local Development

Place the project inside the XAMPP htdocs directory, start Apache, and open:

    http://localhost/nobles-portfolio/nobles-portfolio/

For Apache on port 8080, open:

    http://localhost:8080/nobles-portfolio/nobles-portfolio/
